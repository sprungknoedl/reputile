Why reputile?
reputile is a platform that combines many different sources for reputation checks into a single downloadable and queryable format. The database can be retrieved in total or filtered based on user defined criterias.

Because reputile does the hard work of collecting all those blacklists and translating them into one format, you can focus on more important things!

What format does your database use?
The format strives to be the most simple to parse and work with. This enables you to use a wide variety of languages and tools. Because a lot of reputation sources allready use some form of CSV format, reputile uses CSV (to be more precise RFC4180) as well. The the columns of our database are: 
source,domain,ip4,last,category,description

How can I filter the results?
The reputile database can be filtered with URL query parameters. Each provided query paramter specifies a condition a row must satisfy. A query to only return malicious domains or IPs is: 
?category=malware
