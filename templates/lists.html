The following blacklist sources are currently integrated into reputile. If you know of any others or want a description changed, please create an issue on github for it.

abuse.ch - ZeuS, Feodo & Ransomware Tracker
abuse.ch tracks Command & Control servers for ZeuS and Feodo trojan and Ransomware around the world and provides you a domain- and a IP-blocklist.

abuse.ch   database

AutoShun
AutoShun is a Snort plugin that allows you to send your Snort IDS logs to a centralized server that will correlate attacks from your sensor logs with other snort sensors, honeypots, and mail filters from around the world.

autoshun.org   database

badips.com
badips.com is a community based IP blacklist service. You can report malicious IPs to badips.com and you can download blacklists or query their API to find out if a IP is listed.

badips.com   database

Bambenek Consulting OSINT
Bambenek Consulting is an IT consulting firm focused on cybersecurity and cybercrime. Every day, there is another story about another company having their banking accounts drained, someone having their identity stolen, or critical infrastructure being taken offline by hostile entities. Led by IT security expert, John Bambenek, we have the resources to bring to your business so you can be sure your organization and your customers’ data is safe. And when disaster does strike, you know you can count on us to be with you every step of the way as you recover from an incident.

bambenekconsulting.com   database

blocklist.de
www.blocklist.de is a free and voluntary service provided by a Fraud/Abuse-specialist, whose servers are often attacked on SSH-, Mail-Login-, FTP-, Webserver- and other services. The mission is to report all attacks to the abuse deparments of the infected PCs/servers to ensure that the responsible provider can inform the customer about the infection and disable them.

blocklist.de   database

BruteForceBlocker
BruteForceBlocker is a perl script, that works along with pf. Its main purpose is to block SSH bruteforce attacks via firewall. When this script is running, it checks sshd logs from syslog and looks for failed login attempts and counts the number of such attempts. When a given IP reaches the configured limit of fails, the script puts this IP to the pf’s table and block any further traffic to the that box from the given IP.

danger.rulez.sk   database

CI Army List
The CI Army list is a subset of the CINS Active Threat Intelligence ruleset, and consists of IP addresses that meet two basic criteria: 1) The IP's recent Rogue Packet score factor is very poor, and 2) The InfoSec community has not yet identified the IP as malicious.

cinsscore.com   database

malc0de
malc0de is an updated database of domains hosting malicious executables during the last 30 days.

malc0de.com   database

malwaredomainlist
Malware Domain List features a list of malware-related sites plus a discussion forum on new threats.

malwaredomainlist.com   database

DNS-BH – Malware Domain Blocklist
The DNS-BH project creates and maintains a listing of domains that are known to be used to propagate malware and spyware. This project creates the Bind and Windows zone files required to serve fake replies to localhost for any requests to these, thus preventing many spyware installs and reporting.

malwaredomains.com   database

PhishTank
PhishTank is a free community site where anyone can submit, verify, track and share phishing data.

phishtank.com   database
